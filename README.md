# DocVerify MZ - Carimbo Digital de Moçambique 🇲🇿

O **DocVerify MZ** é uma plataforma de custódia e validação de documentos desenhada para eliminar a burocracia física e a falsificação em Moçambique.

## 🎯 Visão do Projeto
Aliando Inteligência Artificial (OCR) à segurança criptográfica, permitimos que cidadãos e empresas organizem documentos críticos (BI, NUÍT, Cartas, Alvarás) num cofre digital seguro.

## ⚖️ Base Legal
O projeto opera em total conformidade com a **Lei nº 3/2017 (Lei das Transações Eletrónicas de Moçambique)**, garantindo que o documento digital e a sua assinatura tenham validade jurídica perante instituições oficiais.

## 🚀 Funcionalidades Principais
- **Cofre Digital:** Organização inteligente com leitura automática (OCR).
- **Selo de Autenticidade:** QR Code único para verificação instantânea via Hash SHA-256.
- **Alertas de Validade:** Notificações para evitar multas de documentos expirados.
- **Pagamentos Integrados:** Ligação com M-Pesa e e-Mola para taxas de renovação.
